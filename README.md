# ✨ 알고리즘 스터디 GitHub 사용 가이드
1. 개인별 폴더를 만들어서(1인당 1개) 문제 풀이 진행
2. 개인별 폴더 안에 주차별 폴더 만들기
3. 주차별 과제를 모두 해결한 후 문제별로 개인 폴더 내 week00 폴더에 업로드
4. 기간 내 문제를 다 풀지 못했더라도 진행한 부분까지 공유해 주세요!
